# gitPull
Experimenting with `git pull`

`git pull` is equivalent to:
* `git fetch`
* `git merge`

When merging:
* `git checkout <receiving branch>`
* `git merge <from_brach>` into the `<receiving_brach>`

Be sure to `get fetch` to get the remote branch info
